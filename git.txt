Git
Centralizado=> Faz upload no momento do arquivo (SVN) controle de versão 
dos arquivos do repositorio de na cog. Commit sempre realizado no servidor


Descentralizado => (git) o commit pode ser dado localmente e depois subir 
para o servidor


3 estagios 
1 - Untracked Files -> Está dentro do reposito, mas não está sendo versionado

2 - Changes to be commited -> pronto para ser comitado

3 -  Arquivo já commitado
